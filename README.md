# 🚀 Space Escape Runner

A simple 2D arcade game built using **React Native** and **Expo**, where players control a spaceship to dodge falling asteroids and achieve the highest score.

---

## 🎮 Features

- 🚀 Control the spaceship using left and right buttons
- ☄️ Randomly falling asteroids
- 💥 Collision detection with Game Over screen
- 🏆 High score saved locally using AsyncStorage
- 🔄 Restart game functionality
- 🌌 Modern UI with gradient background
- 📱 Android application built using Expo

---

## 🛠️ Tech Stack

- React Native
- Expo
- JavaScript (ES6)
- AsyncStorage
- Expo Linear Gradient

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Sreelakshmii-R/SpaceEscapeRunner.git
```

Move into the project:

```bash
cd SpaceEscapeRunner
```

Install dependencies:

```bash
npm install
```

Start the Expo development server:

```bash
npx expo start
```

Run the app using:

- Expo Go
- Android Emulator
- Physical Android Device

---

## 🎯 Gameplay

1. Press **Start Game**.
2. Move the spaceship left or right.
3. Avoid falling asteroids.
4. Each asteroid that safely passes increases your score.
5. If an asteroid collides with the spaceship, the game ends.
6. Beat your highest score and restart to play again.
